About us

Esse projeto foi criado com o framework Springboot, com back-end em Java e front-end em HTML e JavaScript.

Elaborado pelas alunas Julia Gato Heitor e Marjorye Ciardullo Ramirez para a matéria de Desenvolvimento para Servidores I no 4º ciclo do curso de Sistemas para Internet da Fatec Rubens Lara.

O "CineSet" é um site fictício, que permite que o usuário se cadastre, entre em uma conta e, nela, adicione resenhas de filmes. Uma vez adicionado o filme e a resenha, essas são listadas na sua página de filmes, onde o usuário poderá, além de adicionar mais, editar as entradas já existentes em sua lista, como apagá-las.
